# Card

Conteneur visuel pour grouper du contenu. Respecte les tokens (radius, shadow, border, background) et gère le **dark mode**.

<Canvas>
  <Story of={Stories.Basic} />
</Canvas>

## Composition

- `Card`
- `CardHeader`
- `CardTitle`
- `CardDescription`
- `CardContent`
- `CardFooter`

<Canvas>
  <Story of={Stories.Composition} />
</Canvas>

## Accessibilité

- Structure sémantique libre (rôles non imposés)
- Contrastes conformes via tokens
- Pas d'`h1` multiple : utiliser `CardTitle` sous un seul `h1` par page
