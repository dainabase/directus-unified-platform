## Non-Breaking Changes

## New Component APIs

### Avatar
```typescript
import { Avatar, AvatarImage, AvatarFallback } from '@dainabase/ui';

<Avatar>
  <AvatarImage src="/avatar.jpg" />
  <AvatarFallback>JD</AvatarFallback>
</Avatar>
```

### Badge
```typescript
import { Badge } from '@dainabase/ui';

<Badge variant="success">Active</Badge>
<Badge variant="warning">Pending</Badge>
<Badge variant="destructive">Error</Badge>
```

### Progress

### Skeleton

## Migration Steps

1. **Update package**
```bash
pnpm update @dainabase/ui@0.4.0
```

2. **No required changes**
All existing code remains compatible.

3. **Adopt new components**
Replace custom implementations with the official components.
