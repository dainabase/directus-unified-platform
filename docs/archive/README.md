# Archive Directory

This directory contains all archived documentation files from the root directory.
Files are organized by category for easier navigation.

## Structure

- `context/` - Context and prompt files
- `sessions/` - Session reports and documentation
- `reports/` - Various reports (cleanup, audit, etc.)
- `dashboard/` - Dashboard related documentation
- `guides/` - Various guides and instructions
- `migrations/` - Migration related files
- `misc/` - Miscellaneous documentation

## Note

These files have been archived during the repository cleanup on August 23, 2025.
They are kept for historical reference but may be outdated.
