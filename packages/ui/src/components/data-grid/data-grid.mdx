# DataGrid (Base)

Tableau standard basé sur `@tanstack/react-table`. Pour des jeux de données > 1 000 lignes, utiliser **DataGridAdv** (virtualisé).

## Fonctions

- Tri par colonne
- Filtre global simple
- Pagination de base
- Sélection de lignes (optionnel)

<Canvas>
  <Story of={Stories.Basic} />
</Canvas>

## Quand utiliser DataGridAdv ?

- > 1 500 lignes
- Colonnes reconfigurables à chaud
- Export CSV, densité, visibilité de colonnes

Voir `Data/DataGridAdv` dans Storybook.

## Accessibilité

- Navigation clavier (flèches, tab, shift+tab)
- Focus visible
- `aria-sort` sur en-têtes triables
