# Dialog

Fenêtre modale basée sur Radix Dialog. Focus trap, retour de focus, fermeture via `Esc`, support clavier complet.

<Canvas>
  <Story of={Stories.Basic} />
</Canvas>

## Accessibilité

- Géré par Radix : focus trap + aria-labelledby/aria-describedby
- Fournir un titre (`DialogTitle`) concis
- Éviter les contenus défilants trop longs (préférer `Dialog.Description` + scroll interne)
