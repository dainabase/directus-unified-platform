Guide complet React Hooks

Pierre Martin
15 janvier 2024
15 min de lecture
2.3k vues
(4.9)

À propos de ce guide

Ce guide couvre l'utilisation complète des React Hooks, de useState aux hooks personnalisés. Prérequis : connaissance de base de React et JavaScript ES6.

Introduction

Les Hooks sont une addition à React 16.8. Ils permettent d'utiliser l'état et d'autres fonctionnalités React sans écrire de classe. Les Hooks sont des fonctions qui permettent de "se brancher" sur des fonctionnalités React comme l'état local d'un composant et le cycle de vie depuis des composants fonctionnels.

Hooks de base

useState

Le Hook useState permet d'ajouter l'état local React à des composants fonctionnels.

import React, { useState } from 'react';

function Example() {
  // Déclare une nouvelle variable d'état, qu'on va appeler « count »
  const [count, setCount] = useState(0);

  return (
    <div>
      <p>Vous avez cliqué {count} fois</p>
      <button onClick={() => setCount(count + 1)}>
        Cliquez ici
      </button>
    </div>
  );
}

Important

N'appelez pas les Hooks dans des boucles, des conditions ou des fonctions imbriquées. Utilisez toujours les Hooks au niveau supérieur de votre fonction React.

useEffect

Le Hook d'effet vous permet d'effectuer des effets de bord dans les composants fonctionnels.

import React, { useState, useEffect } from 'react';

function Example() {
  const [count, setCount] = useState(0);

  // Similaire à componentDidMount et componentDidUpdate :
  useEffect(() => {
    // Met à jour le titre du document via l'API du navigateur
    document.title = `Vous avez cliqué ${count} fois`;
    
    // Cleanup function (optionnel)
    return () => {
      console.log('Cleanup');
    };
  }, [count]); // Ne s'exécute que si count change

  return (
    <div>
      <p>Vous avez cliqué {count} fois</p>
      <button onClick={() => setCount(count + 1)}>
        Cliquez ici
      </button>
    </div>
  );
}

Hooks avancés

useContext

Accepte un objet contexte et renvoie la valeur de contexte actuelle.

useReducer

Une alternative à useState. Accepte un réducteur de type (state, action) => newState.

useMemo

Renvoie une valeur mémoïsée. Ne recalcule la valeur que lorsque l'une des dépendances change.

Astuce Pro

Utilisez useMemo pour optimiser des calculs coûteux, mais ne l'utilisez pas partout. La mémorisation a aussi un coût !

Hooks personnalisés

Créer vos propres Hooks vous permet de réutiliser la logique d'état entre différents composants.

// useCounter.js
import { useState } from 'react';

function useCounter(initialValue = 0) {
  const [count, setCount] = useState(initialValue);
  
  const increment = () => setCount(count + 1);
  const decrement = () => setCount(count - 1);
  const reset = () => setCount(initialValue);
  
  return { count, increment, decrement, reset };
}

// Component.js
function Counter() {
  const { count, increment, decrement, reset } = useCounter(0);
  
  return (
    <div>
      <p>Count: {count}</p>
      <button onClick={increment}>+</button>
      <button onClick={decrement}>-</button>
      <button onClick={reset}>Reset</button>
    </div>
  );
}

Conclusion

Les Hooks React offrent une façon plus directe d'utiliser les fonctionnalités React dans les composants fonctionnels. Ils rendent le code plus lisible et plus facile à tester.

Points clés à retenir

  • Les Hooks permettent d'utiliser l'état sans classes
  • Appelez toujours les Hooks au niveau supérieur
  • Utilisez les règles ESLint pour les Hooks
  • Créez des Hooks personnalisés pour réutiliser la logique

Solution Comment gérer plusieurs états avec useState ?
Posté par Sophie L. il y a 2 jours

Je me demande quelle est la meilleure pratique pour gérer plusieurs états dans un composant. Vaut-il mieux utiliser plusieurs useState ou un seul avec un objet ?

Réponse de Pierre Martin (Auteur)

Excellente question ! La règle générale est d'utiliser plusieurs useState pour des valeurs qui changent indépendamment. Si les états sont liés, vous pouvez utiliser un objet ou passer à useReducer.

// ✅ Bon : états indépendants
const [name, setName] = useState('');
const [age, setAge] = useState(0);

// ✅ Bon : états liés
const [user, setUser] = useState({ name: '', age: 0 });
