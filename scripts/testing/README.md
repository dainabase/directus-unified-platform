# Scripts de Test

Ce dossier contient tous les scripts de test du projet :

- **test-*.js** : Tests unitaires et d'intégration
- **check-*.js** : Scripts de vérification
- **validate-*.js** : Scripts de validation
- **diagnose-*.js** : Scripts de diagnostic

## Utilisation

```bash
node scripts/testing/test-connection.js
node scripts/testing/validate-full-system.js
```
